# Open Gallery Prompt Registry

Community-maintained prompt library for [Open Gallery](https://github.com/sdballpark/OpenGallery) — the open source universal AI image studio.

## What is this?

This repository contains the curated prompt registry that powers Open Gallery's built-in prompt library. The `opengallery-prompts.json` file is fetched automatically by the app on startup.

## Categories

- Style Transfer
- Enhancement
- Analysis & Description
- Creative Transformation
- Holiday & Seasonal
- Birthday & Celebration
- Security & Technical

## How to use

In Open Gallery → Settings → Prompt Registry URL, paste the raw URL:

```
https://raw.githubusercontent.com/sdballpark/opengallery-prompts/main/opengallery-prompts.json
```

## Contributing

Have a great prompt? Open a pull request. Add your prompt to the appropriate category in `opengallery-prompts.json` following the existing schema.
